## Cara Pakai

### Prerequisites
- Python 3.x
- pip

### Installation
1. Clone repository
2. Create virtual environment
   - python -m venv venv
3. Activate virtual environment
   - Windows:
     - venv\Scripts\activate
   - Linux/macOS:
     - source venv/bin/activate
4. Install dependencies
   - pip instal -r requirements.txt
   - pip install ntlk
5. Run the project
   - python main.py
6. Deactivate
   - deactivate
